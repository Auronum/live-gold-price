Display live price of Gold, Silver, and Platinum on your WordPress site with customizable themes.

== Description ==

The Live Gold Price Table plugin allows you to effortlessly display the current live prices of Gold, Silver, and Platinum on your WordPress site. You can simply use the provided shortcode [live_gold] in your posts, pages, or widgets to embed the price table.

You can also customize the appearance of the price table by specifying a theme. The available themes are "brand", "dark", and "light". The default theme is "brand". To specify a theme, use the attribute "theme" in the shortcode. For example: [live_gold theme="dark"]

== Installation ==

1. Upload the `live-gold-price-table` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the shortcode [live_gold] in your posts, pages, or widgets to display the live gold price table
4. Customize the appearance by specifying a theme using the attribute "theme" in the shortcode
